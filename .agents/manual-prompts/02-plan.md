# 02 — Plan (spec → plano técnico)

Use para transformar a spec em um plano técnico pequeno.
Copie o bloco abaixo na ferramenta/modelo escolhido.

```text
Leia docs/spec.md, docs/architecture.md e docs/NOW.md.

Tarefa: produzir um plano técnico curto para a próxima entrega, no formato de .agents/templates/plan.md:
- objetivo (1 frase)
- abordagem (passos pequenos, ordenados)
- arquivos/áreas prováveis a tocar
- impacto em arquitetura (se houver) → ADR resumido para docs/architecture.md
- riscos e edge cases
- Definition of Done

Regras:
- KISS. Menor caminho que atende a spec.
- Sem novas abstrações ou stack.
- Se o plano mudar contrato/regra → aponte a atualização necessária em docs/spec.md.
- Devolva o conteúdo completo pronto para colar.
```
