# 03 — Tasks (plano → tarefas pequenas)

Use para quebrar o plano em tarefas pequenas e ordenadas.
Copie o bloco abaixo na ferramenta/modelo escolhido.

```text
Leia o plano técnico (de 02-plan), docs/spec.md e docs/NOW.md.

Tarefa: quebrar o plano em tarefas pequenas e ordenadas, no formato de .agents/templates/tasks.md:
- cada tarefa é pequena, reversível e entregável sozinha
- ordem de execução clara (dependências explícitas)
- critério de pronto por tarefa

Depois: preencha docs/NOW.md apenas com a PRIMEIRA tarefa (objetivo + arquivos prováveis + critério de pronto + o que não fazer).

Regras:
- Uma task por vez no NOW.
- KISS. Não agrupe muita coisa numa tarefa.
- Devolva o conteúdo completo pronto para colar.
```
